# react-example

### Getting Started
1. Install Node.js using [NVM](http://github.com/creationix/nvm) for OSX/Linux or by going to [Nodejs.org](http://nodejs.org) for Windows
2. ```npm install -g zab```
3. ```npm init -y```
4. ```npm install``` <-- installs package dependencies (react, moment, etc.)
5. ```zab watch --open```
